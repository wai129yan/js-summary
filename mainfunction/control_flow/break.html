JavaScript is a control structure used to execute a block of code conditionally.

if (condition) { // Code to execute if the condition is true } else { // Code to execute if the condition is false }

Condition တွေကို left to right စစ်ဆေးပါတယ် Condition တစ်ခုနဲ့ ကိုက်ညီမှသာ code block execute လုပ်ပါတယ် Nested condition များ ရှိနိုင်ပါတယ် Performance အတွက် condition များကို အဓိကထားသော condition က မ complexity နည်းသော condition မှ စပြီး စစ်ဆေးသင့်ပါတယ်
