Remove the last element from an array and returns that element.

let lastElement = array.pop();

pop() က original array ကို modify လုပ်ပါတယ် Empty array မှာ pop() လုပ်ရင် undefined ရပါတယ် pop() က array ရဲ့ နောက်ဆုံး item တစ်ခုတည်းကိုပဲ ဖယ်ထုတ်ပါတယ် pop() ကို သုံးပြီး ဖယ်ထုတ်လိုက်တဲ့ item က array ထဲကနေ အပြီးအပိုင် ပျောက်သွားပါတယ်
