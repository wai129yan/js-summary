sort() သည် JavaScript Array method တစ်ခုဖြစ်ပြီး၊ array ထဲရှိ elements များကို အလိုအလျောက် sorted order တွင် ပြောင်းပေးသည်။ ဒါကြောင့် စီထားတဲ့အတိုင်းတင်မဟုတ်ဘဲ၊ စီထားရမည့် စီစဉ်နည်းကိုသတ်မှတ်ပေးနိုင်ပါသည်။

array.sort(compareFunction);

compareFunction (optional): ထို function သည် array ၏ element နှစ်ခုကို စစ်ဆေး၍၊ elements များကို ဘယ်လိုစီရန် သတ်မှတ်ပေးသည်။ Return Value: sort() သည် sorted array ကို return ပြန်ပေးသည်။
