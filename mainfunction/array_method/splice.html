splice() သည် JavaScript Array method တစ်ခုဖြစ်ပြီး၊ array ထဲမှ element များကို ပြောင်းလဲခြင်း၊အကြောင်းပြုခြင်း သို့မဟုတ် ဖျက်ခြင်းအတွက် အသုံးပြုသည်။ ထို့အပြင်၊ array ၏တည်နေရာတစ်ခုတွင် element များကို ထည့်သွင်းရန်လည်း အသုံးပြုနိုင်ပါသည်။

array.splice(start, deleteCount, item1, item2, ..., itemN);

start: အစပြုသော index ရပ်တည်ချက်။ deleteCount: ဖျက်မည့် element များအရေအတွက် (optional)။ item1, item2, ..., itemN: array ထဲသို့ ထည့်သွင်းရန် element များ (optional)။
