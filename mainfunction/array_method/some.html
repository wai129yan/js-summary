array.some(callback(element, index, array));

some() သည် JavaScript Array method တစ်ခုဖြစ်ပြီး၊ မည်သည့် element မှာမဆို provided callback function ရဲ့ condition ကို ဖြည့်စွက်နိုင်သည်ကို စစ်ဆေးရန်အသုံးပြုသည်။ အကယ်၍ တစ်ခုခုဖြည့်မည်ဆိုပါက true ကို return ပြန်ပေးမည်။ မဖြည့်ပါက false ကို return ပြန်ပေးသည်။ some() ကို ဖုန်းနံပါတ် တစ်ခုရဲ့ validity စစ်ဆေးခြင်း၊ အချိန်ဇယားမှ available slots ရှိ/မရှိ စစ်ဆေးခြင်းစသဖြင့် အများကြီးအသုံးပြုနိုင်သည်။
