extract a shallow copy of a portion of an array without modifying the original array

array.slice(start, end);

slice() က မူလ array/string ကို မပြောင်းလဲစေပါ endIndex မထည့်ရင် array အဆုံးထိယူပါတယ် Negative index တွေသုံးလို့ရပါတယ် (-1 က နောက်ဆုံး item) startIndex သာထည့်ပြီး endIndex မထည့်ရင် startIndex ကနေ အဆုံးထိယူပါတယ် slice() ကို လက်တွေ့အသုံးချနိုင်တဲ့နေရာများ: Array/String တွေကို အပိုင်းလိုက်ခွဲယူချင်တဲ့အခါ Data တွေကို filter လုပ်ချင်တဲ့အခါ Pagination လုပ်တဲ့အခါ နောက်ဆုံး item အချို့ကိုပဲ ပြချင်တဲ့အခါ မူလ data ကို မပျက်စီးစေချင်ဘဲ အပိုင်းလိုက်ယူချင်တဲ့အခါ
