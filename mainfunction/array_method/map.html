map() method is used to iterate over an array and create a new array with the results

let newArray = array.map(function(element, index, array) { });

မူရင်း array ကို မပြောင်းလဲစေပဲ array အသစ်တစ်ခု ရရှိခြင်း array ထဲက item တိုင်းကို တူညီတဲ့ process လုပ်လို့ရခြင်း code ရေးသားရတာ ရှင်းလင်းပြီး နားလည်ရလွယ်ကူခြင်း chaining နဲ့ တခြား array method တွေနဲ့ တွဲသုံးလို့ရခြင်း
